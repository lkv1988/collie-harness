# 记忆元规则（精简参考）

从认知科学、神经科学、复杂系统、认知架构、计算机科学、信息论 6 个领域交叉验证提炼的 7 条基元规则。
灵感来源：Boids 算法——极少的局部规则涌现出全局智能行为。

完整学术证据链见原始文档：`记忆元规则.md`

---

## 7 条元规则

| # | 规则 | 一句话 |
|---|------|--------|
| R1 | 双存储分离（Dual-Store Separation） | 快存储写新经验，慢存储固化结构化知识，二者绝不混用 |
| R2 | 幂律衰减（Power-Law Decay） | 记忆强度幂律衰减（先陡后平），初期 24h 内复习最关键 |
| R3 | 使用即强化（Retrieval = Reinforcement） | 每次检索都强化记忆，主动回忆优于被动复读 |
| R4 | Schema 路由（Schema-Fit Routing） | 高匹配新信息快速同化，低匹配+低显著性则自然衰亡 |
| R5 | 离线整合（Offline Consolidation） | 独立后台进程做压缩/重放/重组织，实时路径只写快存储 |
| R6 | 检索即重写（Retrieval Opens Rewrite Window） | 访问记忆打开有时限的修改窗口，伴随预测误差则更新 |
| R7 | 质量门控（Quality-Gated Entry） | 写入前评估显著性，不改变未来行为的信息不存 |

## 从元规则到实现

```
写入路径：  新信息 → [R7 质量门控] → [R4 Schema 路由] → [R1 快存储]
固化路径：  [R5 离线整合] → 从快存储筛选 → 交错重放 → 写入慢存储
衰减路径：  [R2 幂律衰减] → 长期不用 → 低于检索阈值 → 功能性遗忘
强化路径：  [R3 使用即强化] → 每次检索增加激活值 → 间隔递增
更新路径：  [R6 检索即重写] → 访问 + 预测误差 → 修改窗口 → 更新或覆盖
```
